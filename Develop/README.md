# Creat_README

![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## Description

Written to use modules and create a professional and clear README.md file. Data will be entered through the terminal command line after executing the index.js file. Once the data is received, a markdown file will be created.

## Installation

```bash
git clone https://github.com/sdwizard13/Create_README.md.git
cd Create_README.md
npm install
```

## Usage

nodex index.js  Running this command starts a series of prompts. Answer each prompt to generate a README.MD file in the project root.
